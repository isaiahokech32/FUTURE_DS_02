# FUTURE_DS_02
Social-Media-Ad-Campaign-Performance-Analysis-Dashboard

# About the Task
This project analyzes social media advertising data from Instagram and Facebook Ads Manager to evaluate campaign performance. 
The goal is to build an interactive dashboard that enables businesses to understand how their ads performed, identify high-engagement posts, measure key performance metrics, and improve future campaigns.

# Project Objectives
The objectives of this project are to:

- Analyze overall ad campaign performance
- Identify posts and ads with the highest engagement
- Calculate and interpret Click-Through Rate (CTR) and Return on Investment (ROI)
- Build an interactive dashboard with filters for audience and campaign insights
- Provide actionable recommendations to improve future advertising campaigns

# Tools used
- Power BI for data analysis and dashboard development
- CSV data exported from Facebook and Instagram Ads Manager
- GitHub for documentation and version control
